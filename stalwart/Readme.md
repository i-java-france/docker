# Local Stalwart Docker

This setup runs Stalwart locally with persistent Docker volumes.

## Start

```bash
docker compose up -d --build
```

Open the setup wizard:

```text
http://127.0.0.1:8080/admin
```

Login with:

```text
username: admin
password: ChangeThisStrongPassword123!
```

The fixed password comes from `STALWART_RECOVERY_ADMIN` in `docker-compose.yaml`.
Change it before exposing this server to any network.

## Setup Wizard

For a local test install:

- Server hostname: `mail.example.test`
- Default email domain: `example.test`
- Disable automatic TLS certificates unless this host has real public DNS.
- Storage: keep RocksDB defaults.
- Directory: keep Internal Directory.
- Logging: choose Console.
- DNS: choose Manual DNS Server Management.

After finishing the wizard, restart:

```bash
docker compose restart stalwart
```

Then sign in again at:

```text
http://127.0.0.1:8080/admin
```

Use the permanent administrator email and password shown on the wizard's final screen.

## Useful Commands

```bash
docker compose logs -f stalwart
docker compose restart stalwart
docker compose down
```

To reset the installation completely, remove the volumes too:

```bash
docker compose down -v
```
